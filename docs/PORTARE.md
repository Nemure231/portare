## Install Portare
It's super easy to install it, you just need to login into your github and let's go.

1. Go to [Portare](../main/docs/GITHUB.md) Github, click the `Code` button, then in `HTTPS` tabs to copy the link.
 <img src="https://github.com/Nemure231/portare/blob/main/docs/PORTARE_IMG/p1.png" align="center" />
<br><br>

2. Click the `+` icon in your github navbar, then click `Import Repository`
<img src="https://github.com/Nemure231/portare/blob/main/docs/PORTARE_IMG/p2.png" align="center" />
<br><br>

3. Paste Portare git link in the `Your old repository's clone URL`, fill whatever in your `Repository Name`, then if you don't want people see your code choose `Private`, then click `Begin Import`
<img src="https://github.com/Nemure231/portare/blob/main/docs/PORTARE_IMG/p3.png" align="center" />

4. The import loading will be starting, and then when it finished, click your repository to check it
<img src="https://github.com/Nemure231/portare/blob/main/docs/PORTARE_IMG/p4.png" align="center" />
<br><br>

5. If everything go well, then you can see your repository, yay!
<img src="https://github.com/Nemure231/portare/blob/main/docs/PORTARE_IMG/p5.png" align="center" />
<br><br>
